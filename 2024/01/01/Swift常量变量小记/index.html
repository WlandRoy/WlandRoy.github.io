Swift常量变量小记

发表于 2024-01-01

var 变量

  • 声明一个变量
    var value = 0
  • 变量的值可以改变
    value = 1

    变量只能改变同一种类型的值,不能修改为其他类型的值,如下如果value改为字符串就会报错

    value = "字符" // 报错

let 常量

  • 声明一个常量
    let constValue = 10

    常量一旦赋值就不能改变,否则会改变

    constValue = 2 // 报错

常量变量的几种声明方式

  • 默认值
    var value = 10
let constValue = 20
  • 也可以先定义类型然后在赋值
    var x:Int,y:Double,z:String
let eror:String,constValue:Int
x = 10
y = 2.0
z = "z"
eror = "error"
constValue = 10

e 可以表示10几次方

var value = 1.28175e2
print(value) // 128.175

类型转换

var three = 3
var threeDouble = Double(three)
print(threeDouble) // 3.0
var PI = 3.1415
var p:Int = Int(PI)
print(p) // 3

类型别名

typealias Age = Int
var age:Age = 10
print(age) // 10
